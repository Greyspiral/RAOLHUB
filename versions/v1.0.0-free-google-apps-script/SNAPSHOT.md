# v1.0.0-free-google-apps-script

This snapshot marks the first working free/low-cost Google Apps Script CRM version.

The source files for this version are in the repository root:

- `appsscript.json`
- `src/`
- `docs/`
- `README.md`
- `VERSIONING.md`

Future stable versions should either be tagged as GitHub releases or copied into a new folder under `versions/` when a handoff snapshot is useful.
